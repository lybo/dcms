# CMS

## How to set up the CMS

#### Build fron-end
```
cd front-end
webpack --config webpack.config.production.js
```

#### Create a empty mongod 
cms

#### Run server
```
npm install
node index.js
```

## Dashboard
http://localhost:5000/dashboard/
```
dpd keygen
dpd showkey
```
Copy and paste it to the dashboard
